# 怎么把 AI HOT 朗读器给别人用

**仓库：** https://github.com/shiguang666118/aihot-reader  

**安装包（推荐直接发这个链接）：**  
https://github.com/shiguang666118/aihot-reader/releases/latest

## 方案一：发压缩包 / Release 链接（最快，适合熟人）

已打好包（本地）：

`F:\vibecoding\aihot-reader\dist\aihot-reader-extension-v1.3.2.zip`

线上：

https://github.com/shiguang666118/aihot-reader/releases/tag/v1.3.2

### 你怎么发

- 微信 / 飞书 / 网盘 / 邮件 发出 zip  
- 或放到 GitHub Release

### 对方怎么装（Chrome / Edge 一样）

1. 解压 zip  
2. 打开 `chrome://extensions` 或 `edge://extensions`  
3. 打开 **开发者模式**  
4. **加载已解压的扩展程序**  
5. 选中解压出的 `aihot-reader-extension` 文件夹  
6. 打开 [AI HOT](https://aihot.virxact.com/all) 并刷新  

zip 里有一份 `给别人安装.txt`，可一并转发给对方。

### 局限

- 浏览器会提示「开发者模式扩展」  
- 不能像商店应用一键安装  
- 你更新版本后，对方要重新下 zip 再「重新加载」

---

## 方案二：上架应用商店（适合公开分发）

| 商店 | 地址 | 费用（以官方为准） |
|------|------|-------------------|
| Chrome 网上应用店 | [Chrome Web Store Developer](https://chrome.google.com/webstore/devconsole) | 一次性注册费（常见约 $5） |
| Edge 加载项 | [Partner Center](https://partner.microsoft.com/dashboard) | 一般免费 |

### 大致步骤

1. 注册开发者账号  
2. 用 zip **打包扩展根目录**（含 `manifest.json` 那一层）上传  
3. 填写：名称、简介、截图、隐私权政策（若有联网/存储）  
4. 提交审核，通过后别人可搜索安装  

### 上架前建议补齐

- 隐私说明：会访问 `aihot.virxact.com`、微软语音接口；本地 `storage` 只存音色/语速等偏好  
- 商店截图 1～3 张  
- 确保 `manifest` 权限最小化（当前权限已较克制）

---

## 方案三：GitHub 开源

1. 建仓库，上传 `extension/` 目录  
2. Release 里挂 `aihot-reader-extension-v1.3.0.zip`  
3. README 写安装步骤（同方案一）  

适合会用 Git 的用户，也方便你发更新。

---

## 怎么选

| 场景 | 建议 |
|------|------|
| 先给几个朋友试用 | **方案一发 zip** |
| 要很多人、不想教开发者模式 | **方案二上架** |
| 开源、可迭代 | **方案三 GitHub + zip** |

---

## 更新给别人时

1. 改 `extension/manifest.json` 里的 `version`  
2. 重新打 zip  
3. 通知对方：覆盖文件夹 → 扩展页点 **重新加载** → 刷新 AI HOT 页面
